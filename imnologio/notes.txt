branch chords
Σε αυτο το branch θα δουλεψουμε πανω στις συγχορδιες και συγκεκριμενα πριν το merge με το master branch θα πρεπει να:

1. γραφτει μια ειδικη σελιδα ετσι ωστε να μπορει να γινει copy paste των lyrics μαζι με τις συγχορδιες και στη συνεχεια να περαστουν στο json file

2. τροποποιησω το data.json ετσι ωστε να υπαρχουν και οι συγχορδιες για καθε υμνο

3. φαινονται οι συγχορδιες στο index.html πανω απο τα lyrics του καθε υμνου

4. προστεθει μηχανισμος ετσι ωστε να γινεται transpose των συγχορδιων
